# Project Plan – Mood & Weather To-Do App

## Goal
Build a to-do app that supports task CRUD, mood selection, and weather-based suggestions using a Weather API. The project will be built incrementally across multiple modules.

## Core Features
- Create a task
- Edit a task
- Complete a task
- Delete a task
- Select mood
- Display weather and weather-based suggestions
- Login/authentication (later module)

## Week-by-Week Plan
### Week 01 – Setup (current)
- Create GitHub repo and clone locally
- Set up folder structure: `src/`, `public/`, `docs/`
- Add README + starter files
- Commit regularly

### Week 02 – Static Front End
- Build HTML/CSS UI to match prototype
- Layout: task list, add task, mood selector, weather section

### Week 03 – Logic
- Implement task CRUD
- Update UI dynamically with JavaScript
- Maintain state in memory (initially)

### Week 04 – Backend + API
- Add backend server
- Integrate Weather API
- Connect frontend to backend endpoints
